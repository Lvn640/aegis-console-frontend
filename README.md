# Aegis Console (Frontend)

The secure visualizer for the Aegis-MCP adapter. 

## 🚀 Setup

1. **Clone the Repo:** `git clone https://github.com/Lvn640/aegis-console-frontend.git`
2. **Install Dependencies:** `npm install`
3. **Run the App:** `npm run dev`
4. **Access the Console:** Open `http://localhost:5173/` in your browser.

## 🛡️ Features

- **Real-Time Log Feed:** Streams the agent's internal reasoning and security intercepts.
- **Biometric Approval Proxy:** Syncs with the Auth0 Guardian CIBA push notification.
- **Interactive Terminal:** Custom command-line interface with a `simulate` mode for security demonstrations.
- **Vault Visualizer:** Shows the injection of ephemeral credentials into the agent's memory.
