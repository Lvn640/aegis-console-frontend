# Aegis Console (Frontend)

Secure visualizer for the Aegis-MCP adapter.

## Setup

1. **Clone the Repo:** `git clone https://github.com/Lvn640/aegis-console-frontend.git`
2. **Install Dependencies:** `npm install`
3. **Run the App:** `npm run dev`
4. **Access:** Open `http://localhost:5173/` in your browser.

## Features

- **Log Feed:** Streams agent reasoning and security intercepts.
- **Biometric Approval Proxy:** Syncs with Auth0 Guardian CIBA push.
- **Terminal:** Command-line interface with `simulate` mode.
- **Vault Visualizer:** Displays injected ephemeral credentials.
